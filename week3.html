This week, I reviewed Chapters 1 and 2, especially the content regarding arrays. In PHP, arrays can be written using the shorthand array syntax: 

  
              $vegetables = ["tomato", "corn", "peppers"];
              $days = [1 => 'Monday', 'Tuesday', 'Wednesday'];
              $state = [ "CA" => "California", "NV" => "Nevada", "OR" => "Oregon"];
In the above example, $vegetables has key-value pairs of $vegetables[0] = "tomato", $vegetables[1] = "corn", and $vegetables[2] = "peppers". In $days, we specified that the key starts with 1, so the key-value pairs would be $days[1] = "Monday", $days[2] = "Tuesday", and $days[3] = "Wednesday".

PHP also has a very useful range() function. We can use range() to create an array of numbers. For example 


              $years = range(2021, 2031);
              $age = range(18, 58);
will create an array $years that has the numbers from 2021 to 2031. What is interesting is that the last number is included in the range. This can be different from other languages, and what comes to mind is Python where the a range(0, 10) does not include 10. For PHP, the $age variable here would produce an array including numbers 18 to 58.

To iterate through the array, the foreach loop is very useful. If only the values of the array are needed, then this syntax is used 

              foreach($years as $value) {
                  echo "$value";  
              }
If the keys are needed as well, then this is needed: 
              foreach($state as $key => $value) {
                  echo "$key - $value";
              }
As a sidenote, the range can also be used to create an array of letters such as range('a', 'm') and is_array() returns whether a variable is an array.

There are also quite a few sorting methods that go with arrays. Just four are 

        
                      
              ksort();
              krsort();
              asort();
              arsort();
The "k" stands for key, the "a" stands for value, and "r" indicates descending order. So ksort() sorts by key in ascending order, krsort() sorts by key in descending order, asort() sorts by value in ascending order, and arsort() sorts by value in descending order.

There is also the explode(separator, $string) which returns an array by separating a string by a specified separator. The opposite is implode(glue, $array) which produces a string by joining together the elements of an array by using a glue string.

Another aspect of PHP arrays is that if the keys of an array are strings, you need to use curly braces to echo the value array element. For example, the curly braces here are required 

              echo "The state is {$state['CA']}";

Supplemental Learning

This week I read about the CSS that goes into making tooltips. While visiting various sites such as Youtube or Google, I noticed how stylized tooltips are used prevalently. I think the thing that confused me when learning about tooltips is the use of the position property. The position property often is set to relative or absolute. Relative is relative to the viewport or where it is normally positioned, and absolute is compared to its closest ancestor. So how does this apply to tooltips?

Since the tooltip text occurs in the HTML code after the tooltip container, the tooltip text will have an absolute position that is absolute to the tooltip container. The positioning of the tooltip text uses top, bottom, left and right. How I like to think of this is how far from the top will the tooltip text be at?
