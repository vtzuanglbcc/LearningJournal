I would like to improve on CSS and I tried to incorporate some CSS properties I found on W3Schools into that website.

It was very interesting that single and double quotes mean different things in PHP. Content within singles quotes are not interpreted but instead are displayed literally. For example, in this: 

                
                  $car = "Camry"; 
                  $color = "blue";
                  echo 'My $car is $color.";
                  echo "My $car is $color.";

the first echo statement will display 'My $car is $color.'. Meanwhile, the second echo will display "My Camry is blue".
