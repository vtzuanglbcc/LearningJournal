Group Project

Reflection

The group project creating Coffee Break was an awesome experience in collaborating with fellow classmates and in working on communication and teamwork skills. We had team members in different countries and time zones so it challenged us to find effective ways to communicate. Through this process, we found that Zoom and Discord were both extremely helpful in allowing everyone to be on the same page and to coordinate the tasks that needed to be completed.

In this project, I contributed to the database design, database table creation, and testing to make sure the database query worked. Additionally, I worked on creating the Menu page, the Detailed Menu Item page, the Reviews, an experimental shopping cart, the Add Products, Edit Products, Add Categories, View Categories, and Edit Categories pages. It was a great experience working with the create-read-update-delete system and making sure all those pages worked. In addition, it was fun adding interactivity to the site with JavaScript and CSS.

The code example above is the code written to display the reviews that users can write on a product. After the user creates a review, the database is queried for all the reviews for a specific item and that data is displayed to the user using the PHP code.
